1. HTML

Definición de HTML HTML (por sus siglas en ingles, HyperText Markup Language), es un lenguaje de marcado que nos permite hacer la estructura de una página web. Tenemos que aclarar que HTML no es un lenguaje de programación, pues carece de mecanismos para ejecutar cálculos, repeticiones o condiciones, lo que sí es, es un lenguaje de marcado. Es un lenguaje que le indicará al navegador web qué tipo de elemento es el que está en la estructura de una página web.
